Hello everyone! This is the first post in my “TIL (Today I Learned)” series. Today, I want to share 5 simple terms that are useful to know when working with AI agents.

1. agents.md — This file gives the agent its main job and high-level instructions. It answers: “What should this agent do?”
2. skills.md — This file contains reusable knowledge or steps for doing specific tasks. It answers: “How should this agent do that task?”

For example, agents.md might say, “Help the user write and send a follow-up email.” Skills.md might contain the steps for writing a clear email, choosing a polite tone, and attaching the right file.

3. MCP (Model Context Protocol) — This is a standard way for an agent to connect with external tools, APIs, or data sources.
4. A2A (Agent-to-Agent) — This is a way for one agent to communicate with another agent and share work. Agents can discover each other through an Agent Card.
5. Subagents — These are smaller helper agents connected to a main agent. They can take on part of the work, which makes the process faster and easier to manage.

Thank you for reading! Happy Learning!
