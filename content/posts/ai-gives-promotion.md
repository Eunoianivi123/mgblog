Hello everyone — this is the third post in my “TIL (Today I Learned)” series. Today I want to discuss how AI can give us promotions instead of taking our jobs.

The roles in the company can be represented as a pyramid:

                          / \
                         / C \            <- Lead Leaders
                        /-----\
                       / EXEC  \
                      /---------\
                     /   MGR     \        <- Lead People
                    /-------------\
                   /      SR       \
                  /-----------------\
                 /      EXP          \     <- Own
                /---------------------\
               /      ENTRY            \   <- Do
              /_________________________\

Now, because of AI, the pyramid can be represented as follows:

                           / \
                          / C \
                         /-----\
                        /       \
                       /  EXEC   \
                      /-----------\
                     /    MGR      \
                    /               \
                   /-----------------\
                  /                   \
                 /\        SR        / \
                /  \----------------/   \
               /    \              /     \
              /  AI  \     EXP    /  AI   \
             /        \__________/         \
            /          \ ENTRY  /           \
           /___________ \______/_____________\

Entry-level jobs are minimized, and the people who were at the entry level have mostly been promoted and doing the work of experienced and senior roles.

As productivity increases(with the help of AI, the developer can now write 500 lines of code instead of 100 lines of code), costs decrease and demand rises. As a result, companies will hire the same number of employees while producing more products and innovations.
